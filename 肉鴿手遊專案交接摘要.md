## 2. 核心玩法與系統（最終 C 版，已落地於 `survivor/game.js`）

**核心迴圈**：怪潮持續逼近 → 自動符咒攻擊 → 走位閃避 → 撿魂火(XP)升級 → 三選一選技能 → Boss 節點 → 終局首領 → 結算 → 局外用月塵永久成長。

- **七元素流派**：火 / 水 / 雷 / 毒 / 影 / 聖 / 風，各有「核心符脈 → 三條分支 → 滿級進化」。進化形態 `EVOLUTIONS`：炎王符 / 霜魄符 / 天雷符 / 蝕魂符 / 夜裂符 / 白燈符 / 裂風符，旋刃→月輪刃。
- **技能規則**：主動技能上限 5（`MAX_ACTIVE_SKILLS`），同名再選即升級，Lv.5 進化；被動不佔槽。
- **敵人**：魑魅(ghoul)、咒師(mage)、巨怪(brute)、疾影(skitter，加速近身)、爆靈(bomber，死亡留爆裂危險區)、護衛(warden，給周圍敵人減傷光環)、Boss、終局首領（66%/33% 破階）。
- **局內結構**：四階段（序幕/伏擊/rupture/護陣）+ Boss 預警 + 戰鬥前兆(telegraph，施法前粉色瞄準線/蓄力圈)。
- **三種模式**：記憶培育(default，75 殺) / 狂潮試煉(gauntlet，110 殺) / 月之庭園(garden，需記憶碎片、無 XP、衝分排名)。
- **Roguelite 抉擇**：Boss 戰利品三選一(`BOSS_REWARD_POOL`) / 異變事件三選一(`EVENT_CHOICE_POOL`) / 升級三選一。
- **局外養成**：月塵(moonDust)貨幣、永久修行(`PERMANENT_UPGRADES`)、召喚伙伴(`SUMMONS`，會真正跟隨行動：月影貓靈/符紙小使/鈴音靈/夜蛾影)、長期任務(`MISSIONS`)、主線 5 章(`STORY_CHAPTERS`，會實際影響開局提示/委託/壓力敵人)、記憶碎片(最多 8)。
- **鐵律：全程無自動回血**。血量只能靠提高最大生命、護盾減傷、短暫無敵；初始血量 70/70。

**可讀性/UX 系統（後期大量補強）**：戰鬥追蹤面板、Boss HUD + 邊緣方向指示、威脅摘要(threat)、小地圖雷達、教學任務 checklist、目標羅盤、出擊簡報(briefing)、戰鬥獎章(combat medals)、里程碑橫幅、本局傷害來源統計(damage readout)、build 進度面板、低血暈影/受擊閃白/震動、主選單圖鑑(敵人/流派/規則四分頁)、首頁「今日路線」導引、結算「本局推進/輸出排行/下一步」卡。

**音效**：純 WebAudio 程式合成（無外部音檔），狀態感知動態音樂（lobby/pause/result/battle/danger/boss 六種 mood），M 鍵切換。

## 4. 開發方法論與 QA（已固化）

- **單檔架構**：所有遊戲邏輯集中在 `survivor/game.js`（目前 **8765 行**），新系統一律以「資料函式 + draw 函式 + `render_game_to_text` 輸出 + `debug_*` hook」四件套加入。**刻意不拆模組、不搬框架**，直到測試覆蓋足夠。
- **測試橋接**：`window.render_game_to_text()`（完整 JSON 狀態）、`window.advanceTime(ms)`（確定性步進）、大量 `window.debug_*` 入口。
- **迭代循環**：小步改 → `node --check survivor/game.js` + `npx.cmd tsc --pretty false` → 本地靜態伺服器(`scripts/serve-static.mjs` / `playtest-survivor-loop.mjs`，port 8093/8094) → Playwright/系統 Chrome 截圖肉眼檢查 → 追加 `progress.md` → bump `index.html` 的 `ASSET_VERSION` 快取字串。
- **版本里程碑**＝`ASSET_VERSION` 字串（格式 `<feature>-<YYYYMMDD>`），目前最新 **`combat-readability-20260622`**。
- **已知 gotcha**：(a) PowerShell 不支援 `&&`、中文字串易被轉碼吃掉 → debug helper 改用 ASCII 入口/數字索引；(b) `.codex/skills` 下的 Playwright client 解析不到專案 node_modules → 改用專案自帶 Playwright + 系統 Chrome；(c) Playwright chromium 版本號需用 `executablePath` 指向實際安裝(1223)。
