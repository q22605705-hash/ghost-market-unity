# Ghost Market Hunter - Unity 2D Framework

This folder is a clean Unity 2D rebuild scaffold for the game framework.

Open `UnityProject` in Unity Hub with Unity 2022.3 LTS or newer.

Core implementation included:

- Player input mapping
- Player state machine
- Movement, jump, dash
- Light attack 3-hit combo
- Heavy attack
- Hitstop values
- Knockback values
- Player hurt/invulnerability rules
- Enemy damage hooks
- Room clear conditions
- Save data schema
- Upgrade entry point
- ScriptableObject definitions for weapons, skills, enemies, and rooms

The broken Phaser prototype remains outside this folder and should not be used as the production base.
